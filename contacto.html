Formularios HTML

¿Para que sirven?

Son herramientas que se utilizan para el intercambio de datos entre un cliente y un servidor (interaccion real), existen diferentes tipos de formularios, y se pueden clasificar segun su objetivo. Estan aquellos cuya funcion es obtener datos de un servidor y luego estan aquellos cuya mision es enviar y guardar datos a un servidor

  • Buscadores - Consulta Query(GET)
  • Inscripcion - Registro(POST)
  • Acceso - Inicio Sesion(POST)
  • Contacto - Consulta Web(POST)

¿Como funcionan?

Para utilizarlos, es necesario colocar el contenido que queremos enviar/recibir dentro de una etiqueta <FORM>, en la misma debemos especificar 2 atributos action="..." y method="..."que indican que debe hacerse con la informacion al enviarse, y como se debe llevar a cambo el proceso.

  • ACTION: Antiguamente se utilizaba el valor mailto:usuario@servidor, pero debido a que necesitamos que el cliente cuente con una aplicacion de correo, hoy en se utiliza algun script en lenguaje de servidor como PHP.
  • METHOD: Existen 2 valores posibles para este atributo, por un lado GET prioriza la respuesta del servidor, es menos seguro/rapido y la informacion enviada es visible desde la barra de direcciones. POST por el contrario, es mas seguro/lento debido a que encripta los datos antes de enviarlos, prioriza el envio del cliente y la respuesta puede no ser inmediata.

¿Cuales son sus componentes?

Los formularios se componen de 2 tipos de elementos, por un lado las etiquetas(LABEL) que funcionan como rotulos que indican que datos debemos ingresar o seleccionar; y por el otro estan los campos(INPUT | TEXTAREA | SELECT) que funcionan como areas en las que el usuario puede ingresar o seleccionar datos, cabe destacar que a cada elemento le corresponde un tipo de dato.

  • Campo de Texto (Aa123!#)
  • Campo Numerico (123456)
  • Campo de e-mail (*@*)
  • Campo Contraseña(****)
  • Campo fecha(dd/mm/aaaa)
  • Campo opción Única si | no
  • Campo opción Multiple Arte | Música | Deportes
  • Campo Multilinea
  • Lista Desplegable
